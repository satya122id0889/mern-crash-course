# MERN E-Commerce App

A full-stack e-commerce application built with MongoDB, Express, React, and Node.js.

## Features
- User authentication (JWT)
- Product catalog
- Shopping cart
- Admin dashboard
- REST API backend
- Responsive frontend

## Tech Stack
- Frontend: React, Redux, Axios
- Backend: Node.js, Express
- Database: MongoDB
- Auth: JWT, bcryptjs
